About Page

This web site host some small Python utility web applications.
These utilities are separated into generic utilities and personal utilities.

Personal utilities are applications that are written and targetted to help me only. They are not of any use of anyone else.

Generic utilities are applications that can be of use to anyone who uses for the same goal.

Personal utilities are:

  1. Url dump

    Takes in a file list of urls and dump them into local Sqlite3 db.

  2. Palindrome validator

    Takes in a text string and displays palindromes found.

HCI utilities are:
